# torch-rmve

Torch implementation of the RMVPE pitch extraction model.

The package exposes `RMVEPitchAlgorithm` for loading the model checkpoint and extracting pitch and periodicity from audio.

Model attribution: the bundled/default RMVPE checkpoint comes from [VoiceConversionWebUI](https://huggingface.co/lj1995/VoiceConversionWebUI/resolve/main/rmvpe.pt).
